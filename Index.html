HTML Introducción

¿Qué es HTML?

HTML es el lenguaje de marcado estándar para crear páginas web.

  • HTML significa Hyper Text Markup Language
  • HTML describe la estructura de las páginas web utilizando el marcado
  • Los elementos HTML son los componentes básicos de las páginas HTML.
  • Los elementos HTML están representados por etiquetas
  • Las etiquetas HTML etiquetan partes de contenido como "encabezado", "párrafo", "tabla", etc.
  • Los navegadores no muestran las etiquetas HTML, pero las utilizan para representar el contenido de la página

Estructura básica de un documento HTML

<!DOCTYPE html>
<html lang="es-ES">
<head>
<title>Page Title</title>
<meta charset="utf-8">
</head>
<body>
<h1>Encabezado de la págiana</h1>
<p>Primer párrafo de texto.</p>
</body>
</html>

Explicación de las etiquetas básicas

  1. La etiqueta <!DOCTYPE html>define el tipo de documento como HTML5
  2. La etiqueta <html> define el elemento raíz de una página HTML
  3. El atributo lang="es-ES" especifica el idioma del documento y va dentro de la etiqueta <html>.
  4. La etiqueta <head> define los metadatos sobre el documento.
  5. La etiqueta <title> define un título para el documento HTML.
  6. La etiqueta <meta charset="UTF-8"> define la codificación de caracteres.
  7. La etiqueta <body> define el contenido visible de la página.
  8. La etiqueta <h1> define un encabezado de texto.
  9. La etiqueta <p> define un párrafo.

Etiquetas HTML

Las etiquetas HTML son nombres de elementos rodeados por corchetes angulares:

<nombre de etiqueta> va aquí ... </tagname>

Etiquetas HTML
normalmente vienen en pares como <p> y </p>
Las etiquetas HTML
Estarán formadas por una etiqueta de inicio y una etiqueta de fin
La etiqueta final
Posee una barra inclinada que se inserta antes del nombre de la etiqueta

Consejo: la etiqueta inicial también se denomina etiqueta de apertura y la etiqueta final la etiqueta de cierre .

Navegadores web

El propósito de un navegador web (Chrome, IE, Firefox, Safari) es leer documentos HTML y mostrarlos.
